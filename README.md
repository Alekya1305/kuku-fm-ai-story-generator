## Getting Started

1. Clone the repository:
```bash
git clone https://github.com/yourusername/kuku-fm-ai-story-generator.git
```

2. Install dependencies:
```bash
cd kuku-fm-ai-story-generator
npm install
```

3. Start the development server:
```bash
npm run dev
```
